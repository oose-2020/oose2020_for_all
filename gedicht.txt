 1. Markiere den neuen Dateiinhalt als zu comitten:
    - git add gedicht.txt
 2. Schau was git sich nun gemerkt hat:
    - git status
 3. Füge eine weitere Zeile zu unserem SWT-Gedicht hinzu und speichere es.
 4. Schau was git NUN sagt:
    - git status
 5. Comitte:
    - git commit
 6. Teste wieder mit status, was git jetzt sagt.  
 7. Finde heraus, wie du ALLE durchgeführten Änderungen comitten kannst
